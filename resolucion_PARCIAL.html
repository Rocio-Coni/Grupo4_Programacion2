  1. Se tiene una variable x (no necesariamente temperatura) que depende de la elevaciÃ³n. Se sabe que entre los 1000 y 3000 metros, esta variable se ve reducido en 2 unidades cada 500 metros. Entre los 3000 y 4000 metros, varÃ­a en 0.5 unidades, y a una altitud mayor, su valor es constante. Cree una funciÃ³n que permita obtener el valor de esta variable, Ãºnicamente con el dato de la elevaciÃ³n.
  • El valor de la variable x a 1000 metros es de 81.4 unidades.

SoluciÃ³n:

funcion <- function(h, x = 81.4){
  if (h == 1000){
    var1 = 81.4
    return(var1)
  } else if (h<=3000 & h> 1000){
    var1 = x - 2*((h-1000)%/%500)
    return(var1)
  } else if (h > 3000 & h<=4000){
    var1 = x - 8 - 0.5*((h-3000)%/%500)
    return(var1)
  } else if (h > 4000){
    var1 = x - 9
    return(var1)
  } else {
    print("error!")
  }
}

funcion(1000)
## [1] 81.4
funcion(1500)
## [1] 79.4
funcion(5000)
## [1] 72.4
funcion(999)
## [1] "error!"

data6 <- data4 %>% 
  group_by(DATE = str_sub(DATE, 1, 7)) %>% 
  mutate(MissVal = sum(is.na(temp))*100/n()) %>%
